EBTCalc is a Reverse Polish Notation (RPN) calculator for Windows, OSX, and Linux. EBTCalc users can create custom buttons, programmed in Javascript, using a convenient editor. EBTCalc is open source.

github

EBTCalc uses the following components (and many more):

  • Electron
  • bignumber.js
  • esprima
  • prettier
  • serialize-javascript

App icon by Everaldo Coelho
